선거 후보자 재산신고 보조도구

가상자산 신고 정리기 v2

OCR, 수동 입력, 종목코드 보완, 검증, 신고문구 생성, 엑셀 내보내기를 한 화면에서 처리합니다.

사용방법

OCR은 초안입니다. 제출 전 잔고증명서 원본과 수량, 단가, 종목코드를 반드시 대조하세요.

1. 기본정보

후보자명, 기준일, 잔고 기준, 천원 산정 방식을 입력합니다.

2. 자료 입력

스크린샷 OCR 또는 직접 입력으로 보유 자산을 정리합니다.

3. 검증

누락 체크와 OCR 신뢰도 표시를 보고 원본과 대조합니다.

4. 출력

신고문구 복사, 엑셀, JSON 백업으로 결과를 보관합니다.

개인정보 안내

입력값과 OCR 결과는 이 브라우저의 localStorage에 저장됩니다. 단, 화면 구동을 위해 Tailwind, SheetJS, Tesseract 외부 CDN 스크립트를 불러옵니다.

스크린샷 자동 변환

잔고증명서 스크린샷 또는 복사 텍스트에서 심볼과 수량을 추정합니다.
